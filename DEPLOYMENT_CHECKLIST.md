# قائمة التحقق من النشر ✅

استخدم هذه القائمة للتأكد من إكمال جميع الخطوات قبل وبعد النشر.

## قبل الرفع على Hostinger

### التحضير المحلي
- [ ] تشغيل `bash prepare-deployment.sh`
- [ ] تحديث `.env.production` بمعلومات السيرفر
- [ ] اختبار الموقع محلياً
- [ ] تصدير قاعدة البيانات (`mysqldump` أو phpMyAdmin)
- [ ] مراجعة جميع الملفات المهمة

### معلومات Hostinger المطلوبة
- [ ] رابط لوحة التحكم (hpanel.hostinger.com)
- [ ] اسم المستخدم وكلمة المرور
- [ ] اسم النطاق (Domain)
- [ ] معلومات قاعدة البيانات:
  - [ ] اسم قاعدة البيانات
  - [ ] اسم المستخدم
  - [ ] كلمة المرور
  - [ ] Host (عادة localhost)

---

## أثناء الرفع

### رفع الملفات
- [ ] إنشاء قاعدة بيانات جديدة في Hostinger
- [ ] حفظ معلومات قاعدة البيانات
- [ ] رفع ملف `best-price-deployment.tar.gz`
- [ ] فك ضغط الملفات في `public_html`
- [ ] نقل محتويات المشروع إلى الجذر
- [ ] نقل محتويات `public/` إلى جذر `public_html/`

### الإعدادات
- [ ] رفع ملف `.env.production` كـ `.env`
- [ ] تعديل معلومات قاعدة البيانات في `.env`
- [ ] تعيين `APP_ENV=production`
- [ ] تعيين `APP_DEBUG=false`
- [ ] تحديث `APP_URL` بالنطاق الصحيح

### الأذونات
- [ ] تعيين أذونات `storage/` إلى 755
- [ ] تعيين أذونات `bootstrap/cache/` إلى 755
- [ ] تعيين أذونات `.env` إلى 644

---

## بعد الرفع

### إعداد قاعدة البيانات
- [ ] استيراد قاعدة البيانات عبر phpMyAdmin
- [ ] أو تشغيل `php artisan migrate --force`
- [ ] تشغيل `php artisan db:seed --force`
- [ ] إنشاء حساب المدير الأول

### الأوامر الضرورية
- [ ] `php artisan key:generate`
- [ ] `php artisan storage:link`
- [ ] `php artisan cache:clear`
- [ ] `php artisan config:cache`
- [ ] `php artisan route:cache`
- [ ] `php artisan view:cache`
- [ ] `php artisan optimize`

### إعداد .htaccess
- [ ] التأكد من وجود `.htaccess` في الجذر
- [ ] إعداد Rewrite Rules لـ Laravel
- [ ] إضافة فرض HTTPS

### SSL وHTTPS
- [ ] تثبيت Let's Encrypt SSL
- [ ] فرض HTTPS في `.htaccess`
- [ ] التحقق من عمل الموقع مع HTTPS

### Cron Jobs (إذا لزم)
- [ ] إضافة Cron Job للـ Schedule
- [ ] إضافة Cron Job للـ Queue Workers (إذا لزم)
