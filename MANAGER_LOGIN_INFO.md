# معلومات دخول المدير

## تم حذف جميع حسابات الأدمن القديمة وإنشاء حساب مدير جديد

### معلومات الدخول:

- **البريد الإلكتروني**: `manager@bestprice.ps`
- **كلمة المرور**: `BestPrice@2026!`

### رابط تسجيل الدخول:

```
http://localhost:8000/login
```

أو

```
http://127.0.0.1:8000/login
```

### ملاحظات هامة:

1. هذا هو الحساب الوحيد للمدير في النظام
2. جميع حسابات الأدمن السابقة تم حذفها نهائياً
3. البريد الإلكتروني مؤكد ولا يحتاج إلى تفعيل
4. يرجى تغيير كلمة المرور بعد أول تسجيل دخول
5. احتفظ بهذه المعلومات في مكان آمن
